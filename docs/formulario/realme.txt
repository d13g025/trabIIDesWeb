1. (0,6 pontos) Seguindo os passos da aula, livros indicados e, a leituras recomendadas;

    a. Desenvolva um formulário WEB de sua preferência que contenha os campos exibidos e monte um texto com as seleções do usuário.
    b. O formulário deve ter no mínimo:
        3 input tipo radio ; (redonda)
        5 input tipo checkbox; (caixa)
        2 input texto;
        2 input numéricas;
    c. Além disso o formulário deve mudar as cores com base em combinações de seleção diferentes:

Exemplo: Background da página fica ROSA se o usuário digitar o nome "Maria", digitar um valor maior que 10 em um campo numérico e selecionar todas as checkbox. (Você também pode mudar cor de: text, cor de uma seção ou div, imagem de fundo... Seja criativo!)

    d. Aplique CSS com Bootstrap para posicionamento e tipografia de elementos conforme a aula anterior.

2. (0,5 pontos) Desenvolva uma página profissional (para uma empresa inventada por você). Faça isso em formato de página WEB e estilize com Bootstrap;

A página deve ter seções, menu com links para locais da páginaou outras páginas;
Exemplo de trabalhos desenvolvidos pela empresa;
Catálogo de produtos e/ou serviços;
3. (0,6 pontos) Desenvolva um jogo de perguntas e repostas, ou outro que desejar, utilizando, HTML, animações CSS, e JavaScript, o tema do jogo deve ser criado pelo aluno.

4. (0,3 pontos) Coloque o código desenvolvido em um repositório Github e envie o link do repositório como resposta, publique uma página Github como a página de exemplo: https://costasilvati.github.io/DesWebBasico/index.html , envie como resposta APENAS o LINK da página publicada, ela deve dar acesso a todas as páginas desenvolvidas nas atividades 1 a 3.
